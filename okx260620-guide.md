## 註冊後怎麼買幣？（三種方式）

### 方式一：P2P 買 USDT（★ 最推薦新手）

用本地法幣向平台商戶買 USDT，OKX 託管資金，商家確認收款後放幣。

**操作步驟：**
1. 登入 OKX App 或網頁 → 頂部選【買幣】→【P2P交易】
2. 選擇你想買的幣（建議先買 **USDT**）
3. 選擇法幣種類（CNY / HKD / TWD 等依你地區）
4. 篩選商戶：**高完成率 + 大量好評 + 短時間放行**
5. 輸入購買金額 → 下單
6. 按商戶提供帳戶**直接銀行轉帳/電子支付**（勿備註 crypto 相關字眼）
7. 付款後點【我已付款】→ 等商戶放幣 → USDT 到現貨錢包

⚠️ **安全守則：**
- 只走平台聊天與訂單，不私加 QQ/微信場外
- 付款後不可取消/退款，耐心等放行
- 遇糾紛點【申訴】，OKX 客服介入

---

### 方式二：信用卡 / 借記卡買幣（快捷但貴）

適合小額快速進場。

1. 【買幣】→【信用卡買幣】
2. 選擇法幣 → 選 USDT / BTC / ETH
3. 輸入金額 → 綁卡 → 支付
4. 到帳後可在現貨帳戶查看

特點：到帳快、操作簡單、**手續費高於 P2P**

---

### 方式三：已有幣/USDT → 現貨區換其他幣

適合已有資產轉進 OKX 的用戶。

1. 【資產】→【充值】→ 選幣種（如 USDT）→ 選鏈（通常 TRC20/ERC20）→ 複製地址
2. 從外部錢包/其他所提幣進來
3. 【交易】→【現貨交易】→ 選對 `BTC/USDT` 或 `ETH/USDT`
4. 輸入數量 → 買入
