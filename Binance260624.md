## Binance（幣安）是什麼平台？

Binance 成立於 2017 年，是目前全球交易量最大的加密貨幣交易所之一，核心服務包括：

- **現貨交易**：BTC、ETH、BNB、SOL、XRP、DOGE、USDT 等數百種幣
- **合約交易**：U 本位／幣本位永續與交割合約（有經驗者適用）
- **Binance Earn**：活期、定期、Staking、雙幣投資、Launchpool 質押挖礦
- **Launchpad / Launchpool**：新幣認購與質押挖礦
- **Binance Web3 Wallet**：多鏈非託管錢包（DeFi、NFT、DEX）
- **P2P 交易**（需 KYC，本文不展開操作步驟）
- **策略交易**：現貨網格、合約網格、定投（DCA）

本文重點：**正確開戶 → 入金觀念 → 現貨下單買幣 → 產品簡介**。

## 入金方式簡述（買幣前要先有資金）

| 方式 | 說明 | 備註 |
|----|----|----|
| 信用卡買幣 | 【買幣】→【信用卡買幣】直接換 USDT/BTC/ETH | 手續費較高，適合小額 |
| P2P 交易 | 法幣向認證商家買 USDT（需 KYC） | 最常見，本文不展開步驟 |
| 區塊鏈充值 | 從外部錢包／其他所提幣進來 | 注意選對鏈(TRC20/ERC20等) |
| 第三方入金 | 部分地區支援銀行/電子支付直充 | 依當地開放功能 |

新手最常路徑：**信用卡小額買 USDT → 現貨區換 BTC/ETH** 或 **P2P 買 USDT → 現貨交易**。

## 第一次下單：現貨區買 BTC 或 ETH

假設你現貨帳戶已有 USDT：

1. 登入 Binance App 或網頁 → 【交易】→【現貨交易】
2. 搜尋並選擇交易對：
   - `BTC/USDT`（買比特幣）
   - `ETH/USDT`（買以太幣）
3. 選委託類型：
   - **市價單（Market）**：立即成交，適合新手少量測試
   - **限價單（Limit）**：指定價格掛單，未觸價不成交
4. 輸入欲花費 USDT 數量（或欲購買幣數量）
5. 點【買 BTC】或【買 ETH】→ 成交後幣出現在現貨錢包

⚠️ 現貨無槓桿、無強制平倉，最適合初學者了解市場。

## 開戶後能用什麼功能？

### 現貨交易
基礎買賣，無強制平倉。標準手續費 Maker/Taker 通常 0.10%，BNB 抵扣有折扣。

### 合約交易（⚠️高風險）
U 本位／幣本位永續與交割，全倉／逐倉可切。  
**新手不建議直接開槓桿**，先充分了解強平機制再考慮。

### Binance Earn（理財）
- **活期**：隨存隨取
- **定期 Locked Staking**：鎖倉拿較高年化
- **Launchpool**：質押 BNB／指定幣挖新幣
- **雙幣投資**：結構化收益產品（漲跌判斷）

> ⚠️ 理財產品有市場與贖回規則，參與前詳閱說明書。

### Binance Web3 Wallet
內建於 App，建立／匯入錢包後可：
- 切換多鏈（EVM／Solana／Bitcoin L2 等）
- 接入 DEX 聚合器兌幣
- 瀏覽 NFT 與 DeFi 協議  
是從 CEX 走向鏈上世界的好起點。
